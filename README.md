# Property Lead Generator

A lightweight web tool designed to help real estate agents and property managers **collect, organize, and manage leads** from potential buyers and sellers. This tool is easy to deploy and works directly in the browser.

## Features
- Simple and intuitive lead collection form
- Fields for name, contact info, and property preferences
- Dropdowns for property type, location, and budget
- Generates a structured lead output for easy follow-up

## How to Use
1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Fill in the lead form to start collecting property leads.

## Deployment
- Works locally or can be deployed on GitHub Pages, Netlify, or any static hosting platform.
- Example GitHub Pages link: `https://yourusername.github.io/your-repo-name/`
